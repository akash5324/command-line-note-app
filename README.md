# command-line-note-app
 app to add delete Read and show the  items added by user.

# welcome to Notes App
use these commands to use the functionality of note app:

1. To Add new Note

node app.js add --title="your Title" --body="your note"

2. To list down all Notes: 

node app.js list

3. To read Note with title:

node app.js read --title="existing or known title"

4. To remove existing Note: 

node app.js remove --title=" your notes title which you wanna remove"
